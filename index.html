Basketball Statistic Record

This was my final performance task during my 2nd semester in first-year college. Built in Java with a GUI interface, the project was designed to record and manage basketball player statistics in real time.

Bella Ciao Cafe Ordering System

This project was developed for our 1st Semester Final Performance Task in college. The goal was to create a working system for a real business in our town. We partnered with Bella Ciao Cafe, a local shop known for its aesthetic and professional ambiance.

After gathering the necessary data from the cafe, my partner created a detailed UI/UX mockup and prototype based on the system's specific needs‚Äîincluding colors, fonts, and layout structure. I then translated the design into functional code using Java.

The system is designed for cashiers to easily list, find, remove and track customer orders more efficiently than traditional handwritten notes. I used CRUD functionality using ArrayList and followed Object-Oriented Programming (OOP) principles, as required by our project rubrics. I also used conditions and operators to handle price calculations and change output, and integrated file handling to generate order receipt, helping both the cashier and customers track and organize transactions in real time.

Kopra Price Conversion

This was my first project as a freelance Java developer, created for a client who needed a console-based program as part of their school requirements. Since the project didn't require a user interface, I used ASCII design elements to make the console layout more visually appealing and structured.

The system calculates the price of kopra, whether it's still on the tree or already processed, and includes a feature to compute the profit-sharing percentage between farmers and their laborers using conditions and operations. Furthermore, I used OOP principle to make the code clean, organized, and easy to maintain. Additionally, I implemented file handling to generate service receipts for each transaction, making the system more practical and professional even with only in console-based system.

This program is especially useful for kopra who want a simple, efficient way to handle pricing and profit distribution.

Eggs Flowershop Inventory

This project was developed for a Fiverr client who needed an ordering system for their flower shop. The client provided a mockup prototype and a specific hex color palette to match the desired design. I built the system using Java with a graphical user interface (GUI), as requested.

The program allows the cashier to manage product availability, input customer orders based on product type and floral design, and automatically calculate total costs. I implemented full CRUD functionality using ArrayList to easily add, track, and update products. Conditional statements and operators were used to calculate and validate orders efficiently. Additionally, I added an order receipt feature using file handling, giving the system a more professional touch. This solution significantly improves the ordering process compared to traditional paper-based methods.
